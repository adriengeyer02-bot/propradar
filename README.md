# PropRadar

**Le comparateur de prop firms qui dit la vérité.**

Données indépendantes • Zéro affiliation cachée • Scoring de fiabilité basé sur les faits.

## 🚀 Live Demo

https://propradar.vercel.app (déploiement en cours)

## Fonctionnalités

- **Comparateur interactif** avec filtres avancés (style de trading, drawdown, budget, news trading, etc.)
- **Scoring PropRadar** sur 100 points (ancienneté, transparence, payouts prouvés, incidents, etc.)
- **Fiches détaillées** pour chaque firm avec historique et alertes
- **Blacklist & Alertes** des firms fermées ou à risque
- **Guide** pour éviter les arnaques

## Stack

- Next.js 16 + TypeScript
- Tailwind CSS + design sombre professionnel (inspiration Terminal Finance / Coinglass)
- 100% client-side filtering & sorting

## Lancement local

```bash
npm install
npm run dev
```

Ouvre http://localhost:3000

## Déploiement

Le projet est optimisé pour Vercel (Next.js natif).

## Données

Les données initiales incluent :

- Firms fermées : My Forex Funds, The Funded Trader, Fidelcrest, SurgeTrader
